# Java Task Manager

A simple console-based task manager written in Java.

## Features
- Add tasks
- View tasks
- Update task status
- Delete tasks

## Tech
- Java
- ArrayList
- Scanner

## Why this project?
Built as a learning exercise to practice Java fundamentals, collections, and program structure.

## Future improvements
- Use a Task class instead of parallel lists
- Input validation
- Persist tasks to a file
